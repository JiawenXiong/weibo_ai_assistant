# 微博AI助手

[English](./README_EN.md) | 简体中文

一个浏览器扩展，为微博添加 AI 分析功能，支持自定义提示词和多模型负载均衡。

## 功能特性

- 🔘 **AI 分析按钮** - 为每条微博自动添加「AI分析」按钮
- 🤖 **多模型支持** - 支持配置多个模型，随机选择实现负载均衡
- 🔄 **模型轮换** - 智能排除上次使用的模型，确保每次分析使用不同模型
- 🔁 **重新分析** - 点击「重新分析」按钮可生成新版本的分析结果
- ✏️ **自定义提示词** - 通过设置页面配置分析提示词
- 📋 **一键复制** - 分析结果支持一键复制到剪贴板
- 🎨 **界面适配** - 适配微博新版界面，支持虚拟滚动
- 🌙 **暗色模式** - 自动适配系统暗色模式

## 安装

### Chrome / Edge

1. 下载或克隆本项目到本地
2. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目文件夹
5. 点击扩展图标，进入设置页面配置 API

## 配置

点击扩展图标进入设置页面，配置以下内容：

| 配置项 | 说明 |
|--------|------|
| API 地址 | LLM API 的完整地址（如 OpenAI 兼容格式） |
| API Key | API 密钥（部分 API 可能不需要） |
| 模型名称 | 支持多个模型，每行一个，随机选择 |
| 提示词模板 | 自定义分析提示词，使用 `{content}` 作为占位符 |

### 模型配置示例

```
gpt-3.5-turbo
gpt-4
claude-3-sonnet
deepseek-chat
```

每次调用时会随机选择一个模型，实现负载均衡。同时，系统会记录上次使用的模型，下次随机选择时自动排除，确保连续分析使用不同模型，获得多样化的分析视角。

### 提示词模板示例

```
请分析以下微博内容，提取关键信息并给出你的观点：

{content}

请从以下角度分析：
1. 主要观点
2. 情感倾向
3. 可能的争议点
```

## 使用方法

1. 访问 [微博](https://weibo.com) 任意页面
2. 每条微博的操作栏会出现「AI分析」按钮
3. 点击按钮，等待 AI 分析完成
4. 分析结果会显示在微博内容下方
5. 点击「复制」按钮可一键复制分析结果
6. 如需重新分析，点击「重新分析」按钮，系统会使用不同的模型生成新的分析结果

## 技术架构

```
weibo_ai_assistant/
├── manifest.json      # 扩展配置（Manifest V3）
├── content.js         # 内容脚本（注入按钮、处理交互）
├── background.js      # 后台服务（API 调用、模型选择）
├── options.html/js    # 设置页面
├── popup.html/js      # 弹窗页面
├── styles.css         # 样式文件
└── icons/             # 扩展图标
```

### 关键技术点

- **Manifest V3** - 使用最新的浏览器扩展规范
- **虚拟滚动适配** - 处理微博 Vue 虚拟滚动列表的 DOM 复用问题
- **MutationObserver** - 监听动态加载的微博内容
- **OpenAI 兼容 API** - 支持各类 OpenAI 格式兼容的 LLM API
